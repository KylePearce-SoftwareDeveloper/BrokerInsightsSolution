Instructions to run project:

1. Using XAMPP start the Apache server and MySQL data base.
2. Click admin on MySQL data base to open phpmyadmin.
3. In phpmyadmin create database "achmebroker" and add to it starting data provided in the specification file resulting in "policies" table
4. Clone sorce code from GitHub into XAMPP server folder, for me "C:\xampp\htdocs\"
5. Type in Chrome browser "http://localhost/BrokerInsightsSolution/html/index.html"

After these 5 steps you should be able to intereact with the solution, adding, editing/updating and deleting data from the Achme Broker Ltd policies table.
